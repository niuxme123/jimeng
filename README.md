# 即梦AI自动化工具

自动化即梦AI (Dreamina) 国际版的 Electron 工具。

## 功能特性

- 📁 **账号管理**: 从文件加载账号，支持邮箱密码登录
- 🎬 **视频生成**: 自动设置参数 (AI Video → Omni reference → Seedance 2.0 Fast → 9:16 → 15s)
- 📤 **素材上传**: 自动匹配文案中的场景、道具、人物，上传对应素材
- 🔗 **智能引用**: 自动在文案中插入 `@Image` 和 `@Audio` 引用
- 📥 **视频拦截**: 自动拦截生成的视频下载链接

## 安装

```bash
cd jmzr
npm install
```

## 运行

```bash
npm start
```

## 使用说明

1. **加载账号**: 点击"加载账号文件"选择账号列表
2. **选择素材**: 点击"选择素材文件夹"选择包含图片/音频的文件夹
3. **输入文案**: 在文本框中输入视频文案，格式示例：
   ```
   场景为街道转角，道具为除渍泡泡乐, 人物为阿俊
   ```
4. **提交任务**: 点击"提交生成任务"，自动完成：
   - 上传匹配的素材
   - 填充文案并插入引用
   - 设置生成参数

## 文件格式

### 账号文件格式
```
邮箱----密码----token----cookies----国家----积分----其他----时间
```

### 素材文件命名
素材文件名应与文案中的名称一致：
- `街道转角.png` → 匹配 "场景为街道转角"
- `除渍泡泡乐.png` → 匹配 "道具为除渍泡泡乐"
- `阿俊.png` + `阿俊.MP3` → 匹配 "人物为阿俊"
